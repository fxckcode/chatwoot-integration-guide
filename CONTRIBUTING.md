# Contributing

Thanks for your interest in contributing to the Chatwoot Integration Guide.

## Development

1. Fork the repository
2. Clone your fork: `git clone https://github.com/your-user/chatwoot-skills.git`
3. Create a branch: `git checkout -b feat/my-contribution`

## Conventions

- **Commits**: [Conventional Commits](https://www.conventionalcommits.org/)
  - `docs: add WhatsApp channel configuration guide`
  - `feat: add webhook integration example`
  - `fix: correct Slack integration steps`
  - `chore: update dependencies`
- **Markdown**: Standard formatting with code blocks where applicable
- **Files**: kebab-case, one article per file

## Pull Request Process

1. Update README.md if you add a new category
2. Ensure all links work
3. Update SKILL.md if the changes affect the skill
4. CI must pass (markdownlint)
5. Request a review

## Reporting Issues

- Use GitHub Issue templates
- Bug: describe the documentation error and suggest a correction
- Feature: describe what to add and why it's useful
