# RULES.md — Chatwoot Integration Guide

## Naming

- Files: `kebab-case.md`
- Categories: lowercase with hyphens
- All identifiers in English

## Content

- Each article is a standalone markdown file
- Frontmatter: title, category, description
- Code blocks with language tags
- Absolute URLs for cross-references

## Git

- Conventional commits: `feat:`, `fix:`, `docs:`, `chore:`
- No AI attribution
- No co-author tags

## Quality

- Verify links before committing
- Keep articles focused on one topic
- Include practical code examples
