# How to read Overview Reports (realtime)?

The Overview page is a real-time update of what is happening in your account––how many conversations are open, which agents are available, etc.

**To view the Overview Report, go to Reports → Overview.** This is what it looks like:

## Metrics in the Overview Report

### Open conversations

This shows the number of conversations on your account that are currently:

* **Open**: Ongoing customer interactions

* **Unattended**: Unattended customer conversations

* **Unassigned**: Conversations waiting to be assigned an agent.

![](https://app.chatwoot.com/rails/active_storage/blobs/redirect/eyJfcmFpbHMiOnsibWVzc2FnZSI6IkJBaHBBd0Y2VHc9PSIsImV4cCI6bnVsbCwicHVyIjoiYmxvYl9pZCJ9fQ==--24a00b3b840a74c7c4bf8cc3f9a366659c6ffb04/live%20view%20of%20open%20conversations.png)

### Agent status

This shows the availability status of agents, categorized by:

* Online

* Busy

* Offline

![](https://app.chatwoot.com/rails/active_storage/blobs/redirect/eyJfcmFpbHMiOnsibWVzc2FnZSI6IkJBaHBBL2g1VHc9PSIsImV4cCI6bnVsbCwicHVyIjoiYmxvYl9pZCJ9fQ==--3f7d844247fe77b4f8da4598b8887e282e6967ae/real%20time%20agent%20status%20update.png)

### Conversation traffic

This heatmap shows what times of the day are the busiest for the last 7 days. Each cell represents the number of conversations received as a color shade; hovering over a cell will show the number of conversations received for that hour of the day.

![](https://app.chatwoot.com/rails/active_storage/blobs/redirect/eyJfcmFpbHMiOnsibWVzc2FnZSI6IkJBaHBBMEVtVUE9PSIsImV4cCI6bnVsbCwicHVyIjoiYmxvYl9pZCJ9fQ==--0b9f07096897b1770f72906d8da49c3162424632/conversation%20traffic%20heatmap%20in%20chatwoot.png)

### Conversations by agents

This shows the number of conversations currently being handled by each agent. It can also be used to monitor their performance and workload.

![](https://app.chatwoot.com/rails/active_storage/blobs/redirect/eyJfcmFpbHMiOnsibWVzc2FnZSI6IkJBaHBBL2w1VHc9PSIsImV4cCI6bnVsbCwicHVyIjoiYmxvYl9pZCJ9fQ==--b3c54cfc25eb562316910645d806c3067559abfc/conversations%20by%20agents.png)
