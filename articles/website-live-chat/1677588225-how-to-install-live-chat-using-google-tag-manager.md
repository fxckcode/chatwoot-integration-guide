# How to install live chat using Google Tag Manager?

If you are using Google Tag Manager on your website, you can install Chatwoot from Google Tag Manager’s dashboard by creating a new tag for Chatwoot. To install the Chatwoot live chat widget, please follow the steps below.

## Step 1. Access the Tag Manager console[**​**](https://www.chatwoot.com/docs/product/channels/live-chat/integrations/google-tag-manager#1-access-the-tag-manager-console "Direct link to heading")

Login to the Tag Manager workspace with your Google account. You will see a workspace similar to the one shown below. Click the "Add a new tag" button from the dashboard.

![](https://app.chatwoot.com/rails/active_storage/blobs/redirect/eyJfcmFpbHMiOnsibWVzc2FnZSI6IkJBaHBBK3R2VHc9PSIsImV4cCI6bnVsbCwicHVyIjoiYmxvYl9pZCJ9fQ==--81df2c5d23eb2a7e8df82c6f3c5facfdabb23520/gtm%20workspace.png)

## Step 2. Create a new tag with Tag Configuration[**​**](https://www.chatwoot.com/docs/product/channels/live-chat/integrations/google-tag-manager#2-create-new-tag "Direct link to heading")

You will be presented with a popup to create a tag. Change the name from "Untitled Tag" to "Chatwoot" or a name you prefer.

Click on tag configuration, and select the "Custom HTML" tag type from the list.

![](https://app.chatwoot.com/rails/active_storage/blobs/redirect/eyJfcmFpbHMiOnsibWVzc2FnZSI6IkJBaHBBL3B2VHc9PSIsImV4cCI6bnVsbCwicHVyIjoiYmxvYl9pZCJ9fQ==--7e97f520cf2d748488c9a420e44b25d30b8b02e1/adding%20a%20chatwoot%20tag%20to%20gtm.png)

Copy the script from the inbox configuration in Chatwoot to the HTML script text area, as shown below. Select the "Support document.write" checkbox.

![](https://app.chatwoot.com/rails/active_storage/blobs/redirect/eyJfcmFpbHMiOnsibWVzc2FnZSI6IkJBaHBBeVZ3VHc9PSIsImV4cCI6bnVsbCwicHVyIjoiYmxvYl9pZCJ9fQ==--4045d8e787a07255b9eb2226f7da9a45bf87fb23/script%20for%20chatwoot%20tag.png)

## Step 3. Setup the Triggering[**​**](https://www.chatwoot.com/docs/product/channels/live-chat/integrations/google-tag-manager#2-create-new-tag "Direct link to heading")

Click "Triggering" and select "All pages" from the list.

![](https://app.chatwoot.com/rails/active_storage/blobs/redirect/eyJfcmFpbHMiOnsibWVzc2FnZSI6IkJBaHBBeUp3VHc9PSIsImV4cCI6bnVsbCwicHVyIjoiYmxvYl9pZCJ9fQ==--f6105e7626599266ffcfa0c7e76f1406cc1aa8d8/trigger%20for%20chatwoot%20tag.png)

Click "Save" to save the tag. You will be able to see the tag listed under Workspace changes. Click on "Submit" to deploy the tag. Provide a version name and deploy the changes by clicking "Publish".

![](https://app.chatwoot.com/rails/active_storage/blobs/redirect/eyJfcmFpbHMiOnsibWVzc2FnZSI6IkJBaHBBelJ3VHc9PSIsImV4cCI6bnVsbCwicHVyIjoiYmxvYl9pZCJ9fQ==--7e6ee0dc359bbb6330e3fcd1343bc0b766a27f8d/chatwoot%20tag%20in%20google%20tag%20manager.png)

## Step 4. Tick it off your to-do's[​](https://www.chatwoot.com/docs/product/channels/live-chat/integrations/google-tag-manager#2-create-new-tag "Direct link to heading")

You have successfully deployed the Chatwoot live chat widget to your website. Refresh your website to see the live chat widget appear on all pages.
