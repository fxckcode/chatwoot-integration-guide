# How to install live chat on a Webflow website?

If your website is created with Webflow and you want to talk to your Webflow website visitors in real-time, you can add the Chatwoot live chat widget to your website. Follow the steps described below.

## Step 1. Create a website inbox in Chatwoot[​](https://www.chatwoot.com/docs/product/channels/live-chat/webflow-integration#1-create-a-website-inbox-in-chatwoot "Direct link to heading")

Please refer to this [guide](https://app.chatwoot.com/hc/chatwoot-user-guide-cloud-version/en/website-live-chat/474) for detailed instructions on setting a website inbox in Chatwoot.

## Step 2. Install the generated script in Webflow[​](https://www.chatwoot.com/docs/product/channels/live-chat/webflow-integration#2-install-the-generated-script-in-webflow "Direct link to heading")

Login to your Webflow account and select your website. Select Settings.

![](https://app.chatwoot.com/rails/active_storage/blobs/redirect/eyJfcmFpbHMiOnsibWVzc2FnZSI6IkJBaHBBN2x4VHc9PSIsImV4cCI6bnVsbCwicHVyIjoiYmxvYl9pZCJ9fQ==--dac19f18722460bebc10eed15a33ad30a12a8ec7/webflow%20settings.png)

[​](https://www.chatwoot.com/docs/product/channels/live-chat/webflow-integration#paste-the-script-in-footer-code-in-webflow "Direct link to heading")Go to **Custom Code** -> **Footer Code**. Paste the script in the "Footer Code". Save the changes.

![](https://app.chatwoot.com/rails/active_storage/blobs/redirect/eyJfcmFpbHMiOnsibWVzc2FnZSI6IkJBaHBBNzF4VHc9PSIsImV4cCI6bnVsbCwicHVyIjoiYmxvYl9pZCJ9fQ==--61e6098dbe7b1e852d3c5a3a567b8ea892796d20/footer%20code.png)

## Step 3. Publish the changes[​](https://www.chatwoot.com/docs/product/channels/live-chat/webflow-integration#3-publish-the-changes "Direct link to heading")

Click on Publish -> Select the website -> Publish to selected domains.

![](https://app.chatwoot.com/rails/active_storage/blobs/redirect/eyJfcmFpbHMiOnsibWVzc2FnZSI6IkJBaHBBOEJ4VHc9PSIsImV4cCI6bnVsbCwicHVyIjoiYmxvYl9pZCJ9fQ==--1a8bfa1af05bac393d76002823964402c72a7740/publish%20changes.png)

## Step 4. Verify

Visit your website and see if the widget appears on the website.
