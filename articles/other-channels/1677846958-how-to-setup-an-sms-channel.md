# How to setup an SMS channel?

You can set up your SMS channel inbox through two providers:

1. Twilio

2. Bandwidth

## Creating an SMS inbox

Follow the steps below.

**Step 1**. Open your Chatwoot dashboard. Go to Settings → Inboxes → Add Inbox.

![](https://app.chatwoot.com/rails/active_storage/blobs/redirect/eyJfcmFpbHMiOnsibWVzc2FnZSI6IkJBaHBBeU9vVHc9PSIsImV4cCI6bnVsbCwicHVyIjoiYmxvYl9pZCJ9fQ==--50e84cc9fa7693b293bcc3413a6d2bbb687bf9e5/adding%20an%20inbox%20in%20chatwoot.png)

**Step 2**. Click on the "SMS" icon.

![](https://app.chatwoot.com/rails/active_storage/blobs/redirect/eyJfcmFpbHMiOnsibWVzc2FnZSI6IkJBaHBBeldvVHc9PSIsImV4cCI6bnVsbCwicHVyIjoiYmxvYl9pZCJ9fQ==--0b6eb4dc181ba51bf96c97ed0463e8aaf04fd5d0/sms%20inbox%20in%20chatwoot.png)

**Step 3**. Choose your API provider and configure the inbox. Explained below are both options.

### Choosing "Twilio" as the API provider

If you want to use Twilio to set up your SMS inbox, choose that from the dropdown. It will display the required fields as explained below.

![](https://app.chatwoot.com/rails/active_storage/blobs/redirect/eyJfcmFpbHMiOnsibWVzc2FnZSI6IkJBaHBBMEdvVHc9PSIsImV4cCI6bnVsbCwicHVyIjoiYmxvYl9pZCJ9fQ==--4128d9ee0e2c33db49199538b892b03fcf68b186/sms-inbox-configuration-screen-chatwoot.png)

These are the inputs required to create this channel:

![](https://app.chatwoot.com/rails/active_storage/blobs/redirect/eyJfcmFpbHMiOnsibWVzc2FnZSI6IkJBaHBBME9vVHc9PSIsImV4cCI6bnVsbCwicHVyIjoiYmxvYl9pZCJ9fQ==--b50ffc81291b3b6b2b4fd4e7909437d2e79b3dd6/Screenshot%202023-03-03%20at%206.12.34%20PM.png)

Click `Create Twilio Channel` after filling in the information.

**Configuring Chatwoot with Twilio Studio**[**​**](https://www.chatwoot.com/docs/product/channels/sms/twilio#configuring-chatwoot-with-twilio-studio "Direct link to heading")

If you use Twilio Studio for a custom conversation flow, updating the webhook URL directly will break your existing integration.

For such cases, you can [navigate to the "Twilio Studio" section in this doc](https://www.chatwoot.com/hc/user-guide/articles/1742910590-how-to-set-up-a-whats_app-channel-with-twilio#faq-s).

### Choosing "Bandwidth" as the API provider

If you want to use Bandwidth to set up your SMS inbox, choose that from the dropdown. It will display the required fields as shown below.

![](https://app.chatwoot.com/rails/active_storage/blobs/redirect/eyJfcmFpbHMiOnsibWVzc2FnZSI6IkJBaHBBMWFvVHc9PSIsImV4cCI6bnVsbCwicHVyIjoiYmxvYl9pZCJ9fQ==--4803abe249ee9cf6c2822387d32d4a27ba0beefa/sms-inbox-setup-bandwidth-chatwoot.png)

Click `Create Bandwidth Channel` after filling in the information.

Once you are done with this step, proceed with adding agents.

**Step 4**. "Add agents" to your inbox.

![](https://app.chatwoot.com/rails/active_storage/blobs/redirect/eyJfcmFpbHMiOnsibWVzc2FnZSI6IkJBaHBBMXFvVHc9PSIsImV4cCI6bnVsbCwicHVyIjoiYmxvYl9pZCJ9fQ==--577b8ddd73b0ba8e73d6cac31011c8e4be2a3aa3/adding%20agents%20to%20a%20chatwoot%20inbox.png)

You have successfully created an SMS inbox. You will start receiving the messages in your Chatwoot dashboard.

Also, see: [Sending an SMS campaign to a group of contacts](https://app.chatwoot.com/hc/chatwoot-user-guide-cloud-version/en/advanced-features-explained/508).
